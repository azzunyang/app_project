## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | 호서대학교 공지사항 |
| 대상 | 호서대학교 재학생 |
| 플랫폼 | iOS · Android (Flutter) |
| 주요 기능 | 공지사항 조회, 카테고리 필터, 검색, 즐겨찾기, AI 챗봇, 푸시 알림 |
| 데이터 소스 | 호서대학교 공식 포털 API (`www.hoseo.ac.kr`) |
| 폴백 | API 실패 시 샘플 데이터 표시 |

### 주요 화면 흐름

```
앱 실행
  └─ SplashScreen (2.2초)
       └─ MainScreen (IndexedStack)
            ├─ [탭 0] NoticeListScreen
            │    └─ NoticeDetailScreen
            ├─ [탭 1] FavoritesScreen
            │    └─ NoticeDetailScreen
            ├─ [탭 2] ChatbotScreen
            └─ [탭 3] SearchScreen
                  └─ NoticeDetailScreen

모든 화면 ──→ SettingsScreen (AppBar 아이콘)
```

#### 레이아웃

```
┌─────────────────────────────┐
│  그라디언트 배경              │
│  (#050E20 → #0F1E3D →        │
│   #1B3E7A, 위→아래)          │
│                              │
│          [여백 2/5]          │
│                              │
│        ┌──────────┐          │
│        │  알림 🔔  │  ← 110px 원형 컨테이너
│        │  + 노란점 │    흰색 12% 투명 배경
│        └──────────┘          │
│                              │
│        호서대학교             │  30px, Bold, letterSpacing 2
│          공지사항             │  16px, white70, letterSpacing 4
│                              │
│          [여백 2/5]          │
│                              │
│         • • •                │  펄싱 로딩 도트 (3개)
│    Hoseo University          │  12px, white38
│                              │
└─────────────────────────────┘
```

#### 애니메이션

| 요소 | 타입 | 구간 | 곡선 |
|------|------|------|------|
| 로고+텍스트 | FadeIn | 0~60% | easeOut |
| 로고 | Scale (0.7→1.0) | 0~60% | elasticOut |
| 텍스트 | SlideUp (20→0px) | 30~80% | easeOut |
| 로딩 도트 | FadeIn | 0~60% | easeOut |
| 도트 각각 | Pulsing Opacity | 반복 (1000ms) | — |

#### 동작

- 총 체류 시간: 2,200ms
- 전환: `FadeTransition` (500ms) → `MainScreen`

### 4.2 공지사항 목록 화면

**파일:** `lib/screens/notice_list_screen.dart`

#### 목적
전체 공지사항을 카테고리별로 탐색, 최신 공지를 배너로 강조

#### 레이아웃 구조

```
┌─────────────────────────────┐
│  AppBar: "호서대학교 공지사항" │  + ⚙️ 설정
├─────────────────────────────┤
│  [오류 배너] (서버 연결 실패 시)│  노란 경고 띠
├─────────────────────────────┤
│  배너 영역 (배경 #0F1E3D)    │
│  ┌───────────────────────┐  │  ClipRRect r=18
│  │  카테고리 그라디언트 카드 │  │  height: 150px
│  │  [아이콘+카테고리] [최신]│  │
│  │  공지 제목 (2줄)        │  │
│  │  부서명         >      │  │
│  └───────────────────────┘  │
│     ●  ○  ○  ○  ○  ○       │  페이지 인디케이터 (3초 자동 슬라이드)
├─────────────────────────────┤
│  카테고리 탭 (흰색 배경)      │  수평 스크롤
│  [전체][학사][장학][취업]...  │  선택: #0F1E3D 채움, 미선택: 테두리
├─────────────────────────────┤
│  공지사항 리스트              │  ListView.builder
│  ┌───────────────────────┐  │
│  │ [📌] 공지 제목 (2줄)   │  │  고정 공지: #F0FDF4 배경
│  │ 👤 부서   📅 날짜      │  │  일반 공지: 흰색 배경
│  │                [카테고리]│  │  카테고리 컬러 뱃지
│  └───────────────────────┘  │
└─────────────────────────────┘
```

#### 배너 컴포넌트 상세

- **PageView**: `padEnds: false`, 3초 자동 슬라이드 (`Timer.periodic`)
- **ClipRRect**: `borderRadius: 18` — 어두운 배경 노출 방지
- **카드 내부**: `Stack` 구조 — 배경 장식 아이콘(투명 80px) + 콘텐츠 Column
- **인디케이터**: 활성 도트 `width: 20`, 비활성 `width: 6`, 높이 `6` — `AnimatedContainer`

#### 카테고리 탭

| 상태 | 배경 | 테두리 | 텍스트 |
|------|------|--------|--------|
| 선택 | `#0F1E3D` | `#0F1E3D` | 흰색, Bold |
| 미선택 | 투명 | `#D1D5DB` | `#6B7280`, Regular |

#### 공지 아이템

- 카드: `borderRadius: 12`, 그림자 5% 검정
- 고정 공지(`isPinned`): 배경 `#F0FDF4` + 📌 아이콘
- 카테고리 뱃지: 카테고리 색상 12% 투명도 배경

#### 데이터 로딩 전략

```
ApiService.fetchAll()
  ├─ 성공: allNotices 업데이트, 알림 마킹
  └─ 실패: sampleNotices 폴백, 오류 배너 표시
```

- `RefreshIndicator`: 당겨서 새로고침

- 이미지 로드: `Referer: https://www.hoseo.ac.kr/` 헤더 필요

### 4.4 챗봇 화면

**파일:** `lib/screens/chatbot_screen.dart`

#### 목적
키워드 기반 FAQ 챗봇으로 학생의 공지 관련 질문에 즉시 응답

#### 레이아웃 구조

```
┌─────────────────────────────┐
│  AppBar                     │
│  [🤖] 호서 도우미            │  32px 원형 아이콘 컨테이너
│       질문하면 답해드려요     │  11px, white70
├─────────────────────────────┤
│  채팅 영역 (reverse ListView) │
│                              │
│  [🤖] 환영 메시지 말풍선      │
│                              │
│  ┌──────────┐ ┌──────────┐  │
│  │🎓 장학금  │ │📅 수강신청│  │  Wrap 형식 소형 칩 (아이콘+텍스트)
│  │   안내   │ │          │  │  탭 시 해당 키워드 즉시 전송
│  └──────────┘ └──────────┘  │
│  ┌──────────┐ ┌──────────┐  │
│  │💼 취업   │ │🤲 사회봉사│  │
│  │   정보   │ │          │  │
│  └──────────┘ └──────────┘  │
│                              │
│  [사용자 말풍선]    →        │  우측 정렬, #0F1E3D 배경
│         ← [봇 말풍선]        │  좌측 정렬, 흰색 배경
│                              │
├─────────────────────────────┤
│  입력창                      │
│  [메시지를 입력하세요...] [▶] │  44px 원형 전송 버튼
└─────────────────────────────┘
```

#### ChatMessage 데이터 모델

```dart
class ChatMessage {
  final String text;       // 메시지 텍스트
  final bool isUser;       // true: 사용자, false: 봇
  final DateTime time;     // 전송 시각
  final List<String>? suggestions;  // 빠른 응답 칩 (첫 메시지만)
}
```

#### 말풍선 스타일

| 타입 | 정렬 | 배경 | 텍스트 | 모서리 |
|------|------|------|--------|--------|
| 사용자 | 우측 | `#0F1E3D` | 흰색 | 오른쪽 하단 4px |
| 봇 | 좌측 | 흰색 | `#1F2937` | 왼쪽 하단 4px |

#### 빠른 응답 칩 (Suggestion Grid)

- 배치: 환영 메시지 말풍선 아래 `Wrap` 레이아웃
- 스타일: 흰색 배경, r=20 pill, 아이콘(15px) + 텍스트
- 항목: 장학금 안내 / 수강신청 / 취업 정보 / 사회봉사

#### 타이핑 인디케이터 (`_TypingDots`)

- 응답 딜레이: 800ms
- 3개 도트, 각 1/3 위상 차이로 페이드 인/아웃 반복 (900ms 주기)
- 색상: `Color.fromRGBO(15, 30, 61, opacity)` — opacity 0.3~1.0

#### 키워드-응답 매핑

| 키워드 | 응답 내용 |
|--------|-----------|
| 장학 | 장학 카테고리 안내 + 국가장학금 신청 링크 |
| 수강 | 포털 수강신청 메뉴 안내 |
| 취업 | 취업 카테고리 안내 + 인턴십 프로그램 |
| 성적 | 성적 이의신청 절차 |
| 봉사 | 사회봉사 카테고리 안내 + 학점 인정 기준 |
| 교양 | 영역별 교양 카테고리 안내 |
| 창업 | 외부 카테고리 + 창업경진대회 안내 |
| 그 외 | 기본 안내 (포털 링크 포함) |

## 5. 데이터 모델

### Notice

```dart
class Notice {
  final String id;          // 고유 ID
  final String title;       // 제목
  final String category;    // 카테고리 (학사/장학/취업/외부/사회봉사/교양)
  final String department;  // 담당 부서
  final String date;        // 날짜 문자열 (yyyy.MM.dd)
  final String? content;    // 본문 내용 (nullable)
  final String? detailUrl;  // 원문 URL (nullable)
  final bool isPinned;      // 고정 공지 여부
  bool isFavorite;          // 즐겨찾기 여부 (mutable)

  static Color categoryColor(String category); // 카테고리 → 뱃지 색상
}
```

## 6. 서비스 레이어

### ApiService

| 메서드 | 반환 | 설명 |
|--------|------|------|
| `fetchAll()` | `Future<List<Notice>>` | 전체 공지 크롤링 |
| `search(query)` | `Future<List<Notice>>` | 키워드 검색 |

- Base URL: `https://www.hoseo.ac.kr`
- User-Agent: iOS Safari 모바일 헤더 (크롤링 우회)
- 타임아웃: 10초

### LiberalDbService

| 메서드 | 반환 | 설명 |
|--------|------|------|
| `loadAll()` | `Future<List<Notice>>` | 교양 과목 DB 조회 |

- `isPinned: true`로 반환 → 교양 탭에서 과목 목록 카드 형식으로 렌더링

### FavoritesService

| 메서드 | 설명 |
|--------|------|
| `applyTo(List<Notice>)` | SharedPreferences → 즐겨찾기 상태 복원 |
| `toggle(Notice)` | 즐겨찾기 토글 + 저장 |
| `clearAll()` | 전체 즐겨찾기 삭제 |

## 7. 알림 시스템

### 구성 요소

| 패키지 | 역할 |
|--------|------|
| `flutter_local_notifications` | 로컬 푸시 알림 표시 |
| `workmanager` | 백그라운드 주기적 작업 |
| `shared_preferences` | 마지막 확인 공지 ID 저장 |

### 동작 흐름

```
앱 시작
  └─ NotificationService.init()        // 알림 채널 초기화
  └─ NotificationService.requestPermission()  // 권한 요청 (iOS)
  └─ Workmanager 등록
       └─ 1시간 주기, 네트워크 연결 시
            └─ NotificationService.checkAndNotify()
                 └─ fetchAll() → 새 공지 감지
                      └─ 새 공지 있으면 → showNotification()
```

### 알림 채널

| 속성 | 값 |
|------|----|
| Channel ID | `hoseo_notices` |
| Channel Name | 호서대 공지사항 |
| 중요도 | High |
| 알림 아이콘 | `@mipmap/ic_launcher` |

### 설정 키 (SharedPreferences)

| 키 | 타입 | 기본값 |
|----|------|--------|
| `push_enabled` | bool | true |
| `cat_{카테고리명}` | bool | 학사/장학: true, 나머지: false |
| `last_seen_id` | String | "" |
