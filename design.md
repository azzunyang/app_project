# 호서대학교 공지사항 앱 (Flutter) 디자인 가이드

## 브랜드 컬러
- Primary (헤더/배너 배경): `#1E3A7B` (진한 네이비 블루)
- Secondary (카테고리 선택): `#2B4EAE`
- Background: `#F4F6FA` (연한 회색)
- Surface (카드): `#FFFFFF`
- Foreground: `#1A1A2E`
- Muted: `#6B7280`
- Tag 중요: `#FF6B6B` (빨강 계열)
- Tag 장학: `#A78BFA` (보라 계열)
- Tag 취업: `#34D399` (초록 계열)
- Tag 학사: `#60A5FA` (파랑 계열)

## 화면 목록

### 1. 홈 (공지사항) 화면
- 상단 헤더: "호서대학교 공지사항" 제목 (네이비 배경, 흰 텍스트)
- 슬라이드 배너: 주요 공지 3개를 자동 슬라이드로 표시 (네이비 카드, 태그, 제목, D-day)
- 카테고리 필터 탭: 전체 / 학사 / 장학 / 취업 / 외부 (수평 스크롤)
- 공지사항 목록: ListView, 각 항목에 제목/부서/날짜/카테고리 태그 표시
- 하단 네비게이션: 4개 탭 (공지, 즐겨찾기, 검색, 설정)

### 2. 즐겨찾기 화면
- 즐겨찾기한 공지사항 목록
- 빈 상태 안내 메시지

### 3. 검색 화면
- 검색 입력창
- 검색 결과 목록

### 4. 설정 화면
- 알림 설정
- 카테고리 구독 설정
- 앱 정보

## 주요 사용자 플로우
1. 홈 → 배너 슬라이드 자동 재생 → 공지 탭하면 상세 화면
2. 홈 → 카테고리 탭 선택 → 필터링된 목록 표시
3. 공지 목록 → 즐겨찾기 버튼 → 즐겨찾기 탭에서 확인
4. 검색 탭 → 키워드 입력 → 결과 목록

## 레이아웃 원칙
- 모바일 세로 (9:16) 기준
- 한 손 조작 최적화
- Material Design 3 준수
- 헤더는 네이비 배경으로 상태바까지 확장
- 카드 그림자 subtle하게 적용
